# 税務・換金チェックリスト（当選額が増えたとき用）

免責: 2026年6月時点の国税庁情報等に基づく一般的整理。最終判断は税理士・所轄税務署に確認を。

---

## 税務（一時所得）チェックリスト

- [ ] 当選は**一時所得**（年50万円特別控除＋超過分の1/2課税）。
- [ ] 物品は**小売価額の60%**で評価、**商品券・ギフト券は券面額**で評価。
- [ ] 経費にできるのは**当選分の応募費用のみ**（ハズレ分は不可）。領収書保存。
- [ ] **給与所得者は年間当選評価額が約90万円超で確定申告が必要**（経費ゼロ・他に副業所得なしの場合）。
      計算: (当選額−経費−50万) × 1/2 が20万円超で申告対象。
- [ ] **所得税が申告不要でも住民税申告は別途必要**（20万円ルールは住民税に不適用）。要市区町村確認。
- [ ] 高額賞金は源泉徴収（50万円控除後10.21%）されても確定申告で精算。支払調書が税務署に届く。
- [ ] **反復・営利・大規模になると一時所得→雑/事業所得に区分変更されるリスク**（優遇が消える）。要税理士確認。

### 当選額の早見
| 年間当選評価額(経費0) | 課税対象(×1/2) | 給与所得者の申告 |
|---|---|---|
| 50万円以下 | 0 | 不要(住民税は別途確認) |
| 90万円 | 20万円 | ほぼ境界。超で申告必要 |
| 150万円 | 50万円 | 申告必要 |

## 売却時チェックリスト

- [ ] 生活用動産（家電・衣服等）の売却は**非課税**。
- [ ] 貴金属・宝石・書画骨董等で**1個/1組30万円超**のみ譲渡所得課税。
- [ ] 反復・営利の売却は雑/事業所得になりうる。

## 換金ルート早見（換金率の高い順）

| 賞品 | 最適ルート | 換金率目安 |
|---|---|---|
| 全国共通商品券(VJA/JCBナイスギフト) | 金券ショップ・宅配買取 | 97〜98% |
| JCBギフト/百貨店共通券 | 金券ショップ | 92〜97% |
| QUOカード | 金券ショップ(ギフト柄・完封が高) | 88〜96% |
| 図書カードNEXT | 金券ショップ(完封で95%) | 約90% |

| 家電・ブランド品 | メルカリ(保証書・型番明記) | 定価60〜80% |
| 特産品・食品 | 自家消費が最善。売るならフリマ少量 | — |

### 換金の実務チェック
- [ ] デジタルコード(Amazon/Apple)は**店頭でなくオンライン買取**を使う。初回特典・手数料無料業者を選ぶ。
- [ ] 物理商品券は**完封・まとめ売り**で数%アップ。贈答需要期(年末・卒入学前)が高い。
- [ ] **複数業者で相見積もり**(店舗間で最大10%差)。
- [ ] **メルカリは2025年10月改定で事業者扱い(月200点/100万円等)はアカウント停止リスク**。当選品売却は「私物整理」の範囲(少量・不定期)に留める。商品券・チケットはメルカリ出品不可→金券ショップへ。
- [ ] 旅行券・宿泊券は記名式・有効期限に注意(期限3ヶ月未満は買取不可のことが多い)。
- [ ] ギフトコードは公開で晒さない(盗用防止)。

## 受け取り時チェック（失効防止・本システムのremindと併用）
- [ ] 当選通知の受取期限・DM返信期限を applog の receive_deadline に記録 → `python toolc.py remind` で通知。
- [ ] デジタルギフトは受領期限内に受領操作(LINEギフトは期限切れ復活不可)。
- [ ] 賞品到着は遅め(平均約38日)。追跡番号で確認、長期不在に注意。
- [ ] カード番号/口座/暗証番号/手数料を求める通知は詐欺(toolcが自動で弾く)。
